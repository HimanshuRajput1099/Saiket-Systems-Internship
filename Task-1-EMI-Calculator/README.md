# Loan EMI Calculator

## Objective
Calculate the monthly EMI for a loan based on principal amount, interest rate, and loan tenure.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User-friendly interface
- Accepts loan amount, interest rate, and tenure
- Calculates EMI instantly
- Displays formatted EMI result

## Formula Used

EMI = (P × R × (1 + R)^N) / ((1 + R)^N - 1)

Where:
- P = Principal Amount
- R = Monthly Interest Rate
- N = Loan Tenure in Months
